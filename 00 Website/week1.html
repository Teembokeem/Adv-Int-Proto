Understanding Processing.

Geting Started

intro

Day one. Processing is a java-based software sketchbook for learning how to code in the context of graphic art. Check it out at processing.org While coding languages like Javascript are event-based programming, processing's primary mode of operation is to run on a loop. During the "prerun", all the code is executed once at the beginning, and will continue to run all other commands in the "loop" on a loop.
